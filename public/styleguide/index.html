Alert messages

@title Alert messages

Alert messages can have different colors accordingly with the message they are comunicating.

Use the following syntax to call an alert message from a view: _showAlert(type, message)

_showAlert('success', 'This is a success message')

In a Layout template (ie. MainLayout.js) use _alert.open(type, message)

this._alert.open('success', 'This is a success message')

Chart container

@title Chart container

The chart-wrapper component is a container for the chartJS generated code. (Documentation here: www.chartjs.org) When this componen is used inside the App-hero some additional styles will be applied.

Brand wrapper

@title Brand wrapper

The brand-wrapper component is a container for the Lif token logo. When this componen is used inside the App-hero some additional styles will be applied.

<h1>
  <Link to="/" className="brand-wrapper">
    <img className="brand" src={lifLogo}/>
  </Link>
</h1>

Card

@title Card

The card component is inherited from bootstrap, but we created custom styles for it. Documentation: https://getbootstrap.com/docs/4.0/components/card/

We added a sub-component called .card-icon which is the small icon placed in the top right corner of the cards. On the home page we used an inline style to center the cards and prevent them to be wider than 300px.

<div className="card ml-lg-4 mr-lg-4 mb-5" style={{maxWidth: 300, margin: '0 auto'}}>
  <div className="card-body text-center">
    <Link
      onMouseDown={()=>$('[data-toggle="tooltip"]').tooltip('hide') }
      to="FAQ" className="card-icon" data-toggle="tooltip" data-placement="top" title="More info...">
      <i className="material-icons link-light">help_outline</i>
    </Link>
    <i className="material-icons material-icons--xl">account_balance_wallet</i>
    <h1 className="h5 card-title text-muted"><strong>MVM Funds</strong></h1>
    <p className="card-text text--xl text-success"><strong>16,278</strong></p>
  </div>
</div>
